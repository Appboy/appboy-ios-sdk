Method Documentation

- (void) allowRequestAlwaysPermission

Calling this method gives Appboy permission to request Always authorization on your behalf the next time we attempt to collect significant location changes.

- (void) allowRequestWhenInUseLocationPermission

Calling this method gives Appboy permission to request WhenInUse authorization on your behalf the next time we attempt to collect location in the foreground.

- (void) logSingleLocation

Calling this method will log a location using the regular location provider if a location is reported in under 60 seconds. After 60 seconds expires the regular location provider will stop collecting location.
