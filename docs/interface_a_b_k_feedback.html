- (ABKFeedbackValidation) feedbackValidation

This method checks the validation of the feedback object and returns the result.

Before submiting a feedback please make sure to call this method to check the validation of the feedback object. A feedback is only valid when it satisfies both of following conditions:

  • feedback message isn't empty;
  • the email is a valid email address.
